快速开始 ​

React File Preview 是一个现代化的 React 文件预览组件，支持多种文件格式的预览。

特性 ​

  • 🖼️ 多格式支持 - 支持图片、视频、音频、PDF、Office 文档、Markdown 和代码文件
  • ⚡ 高性能 - 基于 React 18 和现代浏览器 API
  • 🎨 可定制 - 支持自定义渲染器和主题
  • 📱 响应式 - 完美适配桌面和移动设备
  • 🔧 易于集成 - 简单的 API 设计
  • 📦 TypeScript - 完整的类型定义
